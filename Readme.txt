## Инструкция по обновлению модели

1. Поместите папку win_files в С:\tensorflow\models\research\
2. Замените файл C:\tensorflow\models\research\object_detection\resnet101_webcam.py

## Запуск модели

Запускается модель таким же образом, как и прежде:
1. Открыть Anaconda Prompt и перейти в паку командой 
cd C:\tensorflow\models\research\object_detection
2. Запустить скрипт командой 
python resnet101_webcam.py

### Замечание 
В данной версии можно запускать модель не только на веб-камере, но и на видеофайлах/видеотрансляциях.
Сделать это возможно добавлением аргумента -vs или --videostream к скрипту в п.2.
Например:
python resnet101_webcam.py -vs rtsp://192.168.0.1

После -vs может стоять либо номер видеоустройства(число), либо путь к видеофайлу(абсолютный), либо адрес трансляции. Эти значения указываются без каких-либо дополнительных символов(кавычек и т.д.)
